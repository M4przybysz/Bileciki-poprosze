# Bileciki poproszę!
Rok 2006, Polska. Rynek pracy powoli podnosi się z popiołów, jednak wiele osób wciąż pozostaje bezrobotne.

Wciel się w rolę Karola, świeżo upieczonego konduktora Polskich Kolei Krajowych, próbującego pozbierać się po trudnym okresie. Rozmawiaj z pasażerami, sprawdzaj bilety, pilnuj porządku w pociągu i... nie dopuść, żeby twoi przełożeni dowiedzieli się, gdzie tak naprawdę mieszkasz.

Wystarczy, że będziesz skrupulatnie wykonywał swoje obowiązki, a wszystko będzie w porządku... ale praca konduktora wcale nie jest tak czarno-biała, jak mogłoby się wydawać...
